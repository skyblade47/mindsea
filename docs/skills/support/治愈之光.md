# 治愈之光

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 辅助 |
| 伤害类型 | 魔法 |
| 使用次数 | 3 次 |
| 品质 | 钢语 |
| 基础威力 | 40 |
| MP消耗 | 20 |

## 技能描述

召唤温暖的圣光笼罩自身，快速恢复生命值。治愈效果基于基础威力，恢复量会受到当前最大生命值的限制，不会超出上限。是战斗中保持续航的核心恢复技能。

## 六维评估

| 维度 | 评分 | 说明 |
|------|------|------|
| 威力 (power) | 6 | 恢复量中等，钢语品质下能回复约40%生命值 |
| 范围 (scope) | 2 | 仅作用于自身，单体效果 |
| 持续 (duration) | 3 | 即时生效，无持续效果 |
| 灵活 (flexibility) | 7 | 任何战斗场景都适用，泛用性强 |
| 独特 (uniqueness) | 4 | 标准的治疗技能，机制较为常见 |
| 联动 (synergy) | 5 | 可与护盾类技能配合形成生存组合 |

## 原子指令实现

```typescript
ops: [
  { op: 'set', target: 'self', path: 'variables.healAmount', value: 40 },
  { op: 'add', target: 'self', path: 'hp', value: 40 },
  { op: 'clamp', target: 'self', path: 'hp', min: 0, max: 999 },
  { op: 'log', text: '治愈之光恢复了40点生命值' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **set**: 设置临时变量 `healAmount` 为 40，用于记录本次治疗量
2. **add**: 向自身 `hp` 属性增加 40 点生命值（治疗必须使用add指令）
3. **clamp**: 将自身 `hp` 限制在 0 到 999 之间，防止生命值超出上限
4. **log**: 输出治疗日志，使用 text 参数显示恢复的生命值

## 使用场景

- 生命值低于50%时使用，快速恢复战力
- BOSS战斗的间隙期，利用输出空窗回血
- 持续消耗战中，保持血线安全以应对突发伤害

## 联动效果

- 与反噬护盾联动：护盾吸收伤害的同时保持血量健康，形成双层防护
- 与力量祝福联动：满血状态下可以更激进地输出，无需保守走位
- 与暴击姿态联动：高血量配合爆发输出，快速压制敌人血线
