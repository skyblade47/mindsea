# 疾风步

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 辅助 |
| 伤害类型 | 物理 |
| 使用次数 | 4 次 |
| 品质 | 铁言 |
| 基础威力 | 30 |
| MP消耗 | 12 |

## 技能描述

身形化作疾风，大幅提升自身移动速度和攻击速度。速度提升效果基于基础威力，增幅比例为30%。由于消耗较低且使用次数较多，是铁言品质中机动性最强的辅助技能，适合需要频繁走位或先手出招的战斗场景。

## 六维评估

| 维度 | 评分 | 说明 |
|------|------|------|
| 威力 (power) | 4 | 速度提升本身不直接造成伤害，但间接提升输出频率 |
| 范围 (scope) | 2 | 仅作用于自身，单体增益 |
| 持续 (duration) | 7 | 增益效果持续时间长，4次使用可覆盖大部分战斗 |
| 灵活 (flexibility) | 9 | 几乎所有场景都能受益，泛用性最高的技能之一 |
| 独特 (uniqueness) | 4 | 标准速度buff，机制常见但实用 |
| 联动 (synergy) | 7 | 与多种技能形成良好配合 |

## 原子指令实现

```typescript
ops: [
  { op: 'set', target: 'self', path: 'variables.speedBoost', value: 0.3 },
  { op: 'mul', target: 'self', path: 'speed', value: 1.3 },
  { op: 'log', text: '疾风步发动，速度提升30%' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **set**: 设置临时变量 `speedBoost` 为 0.3，表示30%的速度增幅
2. **mul**: 将自身 `speed` 属性乘以 1.3，实现30%的速度提升
3. **log**: 输出增益日志，使用 text 参数显示速度提升的百分比

## 使用场景

- 需要先手出招的战斗，确保先制攻击
- 面对高速敌人时，弥补速度劣势
- 配合多段攻击技能，提升出手频率
- 战斗中需要频繁调整站位时使用
- 持久战中保持4次使用覆盖全场

## 联动效果

- 与力量祝福联动：高攻速配合高攻击，输出效率大幅提升
- 与暴击姿态联动：更多的攻击次数意味着更多的暴击触发机会
- 与吸血攻击联动：攻击频率提升使吸血回复更稳定
- 与反噬护盾联动：高速可以减少被击中次数，延长护盾存续时间
