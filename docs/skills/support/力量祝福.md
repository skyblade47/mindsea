# 力量祝福

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 辅助 |
| 伤害类型 | 物理 |
| 使用次数 | 3 次 |
| 品质 | 铁言 |
| 基础威力 | 25 |
| MP消耗 | 15 |

## 技能描述

以战魂之力加持自身，短时间内大幅提升攻击力。攻击力提升效果基于基础威力计算，增幅比例为25%。是铁言品质中性价比极高的输出型辅助技能，适合在爆发期使用以最大化伤害收益。

## 六维评估

| 维度 | 评分 | 说明 |
|------|------|------|
| 威力 (power) | 5 | 25%攻击力提升，铁言品质中规中矩 |
| 范围 (scope) | 2 | 仅作用于自身，单体增益 |
| 持续 (duration) | 6 | 增益效果持续至战斗结束或被驱散 |
| 灵活 (flexibility) | 8 | 所有输出场景均适用，泛用性极强 |
| 独特 (uniqueness) | 3 | 标准攻击力buff，机制较为普遍 |
| 联动 (synergy) | 8 | 与各类输出技能联动效果极佳 |

## 原子指令实现

```typescript
ops: [
  { op: 'set', target: 'self', path: 'variables.atkBoost', value: 0.25 },
  { op: 'mul', target: 'self', path: 'attack', value: 1.25 },
  { op: 'log', text: '力量祝福激活，攻击力提升25%' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **set**: 设置临时变量 `atkBoost` 为 0.25，表示25%的攻击力增幅
2. **mul**: 将自身 `attack` 属性乘以 1.25，实现25%的攻击力提升
3. **log**: 输出增益日志，使用 text 参数显示攻击力提升的百分比

## 使用场景

- BOSS战爆发阶段开启，最大化伤害输出
- 面对高防御敌人时，配合穿透效果使用
- 战斗初期叠加增益，为后续输出铺路
- 与暴击姿态同时使用，形成爆发组合

## 联动效果

- 与暴击姿态联动：攻击力提升配合双倍伤害，爆发能力翻倍
- 与吸血攻击联动：高攻击力带来更高的吸血回复量
- 与疾风步联动：高攻速配合高攻击，DPS显著提升
- 与反噬护盾联动：攻击提升使护盾反弹的伤害也相应增加
