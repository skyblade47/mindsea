# 吸血攻击

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 特殊 |
| 伤害类型 | 物理 |
| 使用次数 | 3 次 |
| 品质 | 银句 |
| 基础威力 | 50 |
| MP消耗 | 25 |

## 技能描述

凝聚暗影之力发动攻击，在造成物理伤害的同时，将部分伤害转化为自身生命值恢复。吸血比例为造成伤害的50%，恢复量受最大生命值上限限制。银句品质下兼具输出与续航能力，是持久战的核心技能之一。

| 独特 (uniqueness) | 7 | 攻击与治疗结合的机制，具有独特性 |
| 联动 (synergy) | 8 | 与攻击力buff、暴击等联动收益极高 |

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'enemy', path: 'hp', value: -50 },
  { op: 'add', target: 'self', path: 'hp', value: 25 },
  { op: 'log', text: '吸血攻击！造成50点伤害，回复25点生命。' },
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add**: 对敌人 `hp` 增加 -50（即减少50），造成50点伤害（造成伤害必须使用add指令，value为负数，会触发护盾和伤害日志）
2. **add**: 向自身 `hp` 增加 25 点生命值，实现吸血回复（治疗必须使用add指令，value为正数）
3. **log**: 输出攻击日志，使用 text 参数显示造成的伤害和回复的生命值

## 使用场景

- 持久战中保持血线稳定，减少治疗技能消耗
- 生命值偏低但需要继续输出时使用
- 面对低攻击高血量敌人时，实现无伤消耗
- BOSS战的平稳期，一边输出一边维持状态

## 联动效果

- 与力量祝福联动：攻击力提升使伤害和吸血量同步增加
- 与暴击姿态联动：暴击双倍伤害带来双倍吸血回复
- 与疾风步联动：更多出手次数意味着更多吸血机会
- 与治愈之光联动：双重恢复机制，生存能力拉满
- 与反噬护盾联动：护盾+吸血形成全方位生存体系
