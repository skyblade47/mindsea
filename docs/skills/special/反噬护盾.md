# 反噬护盾

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 特殊 |
| 伤害类型 | 魔法 |
| 使用次数 | 2 次 |
| 品质 | 源章 |
| 基础威力 | 60 |
| MP消耗 | 35 |

## 技能描述

召唤源力护盾笼罩自身，吸收伤害的同时将部分伤害反弹给攻击者。护盾值为60点，反弹比例为受到伤害的30%。护盾存在期间，所有伤害优先由护盾吸收，护盾值耗尽后才会扣减生命值。源章品质的防御型技能，兼具生存与反制能力，是面对高攻击敌人时的强力手段。

| 灵活 (flexibility) | 7 | 面对物理或魔法攻击都有效 |

| 联动 (synergy) | 6 | 与治疗、速度类技能有不错的配合 |

## 原子指令实现

```typescript
ops: [
  { op: 'set', target: 'self', path: 'shield', value: 60 },
  { op: 'mul', target: 'enemy', path: 'attack', value: 0.7 },
  { op: 'log', text: '反噬护盾激活！获得60点护盾，敌人攻击力降低30%' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **set**: 设置自身 `shield` 属性为 60，获得60点护盾值
2. **mul**: 将敌人 `attack` 属性乘以 0.7，实现降低30%攻击力的效果（简化的反噬机制，通过削弱敌方攻击来体现护盾的反制能力）
3. **log**: 输出护盾激活日志，使用 text 参数显示获得的护盾值和敌方减攻效果

## 使用场景

- 面对高攻击BOSS时，有效降低受到的伤害
- 敌人即将释放大招前开启，抵消关键伤害
- 持久战中减少治疗压力，同时反伤消耗敌人
- 配合吸血攻击，形成攻防循环的持久战体系
- 低血量时作为保命手段，争取喘息空间

## 联动效果

- 与治愈之光联动：护盾吸收+生命恢复，双重生存保障
- 与力量祝福联动：攻击力提升使反弹的伤害也相应增加
- 与疾风步联动：高速减少被击中次数，延长护盾存续时间
- 与暴击姿态联动：反弹伤害虽然不触发暴击，但护盾保障生存让你能等到爆发窗口
- 与吸血攻击联动：护盾扛伤+攻击吸血，生存能力极强
