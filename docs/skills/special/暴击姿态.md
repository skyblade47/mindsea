# 暴击姿态

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 特殊 |
| 伤害类型 | 物理 |
| 使用次数 | 2 次 |
| 品质 | 银句 |
| 基础威力 | 0 |
| MP消耗 | 20 |

## 技能描述

进入高度专注的战斗姿态，使下一次攻击必定造成双倍伤害。暴击状态存储于变量中，在下一次攻击命中后消耗。银句品质下的爆发型技能，虽然使用次数较少，但配合高威力攻击技能可以瞬间造成毁灭性伤害，是打出爆发输出的核心技能。

## 六维评估

| 维度 | 评分 | 说明 |
|------|------|------|
| 威力 (power) | 9 | 双倍伤害的提升幅度极大，银句品质顶级爆发 |
| 范围 (scope) | 2 | 仅作用于自身的状态，依赖后续攻击技能 |
| 持续 (duration) | 5 | 持续到下一次攻击，时间不定但效果确定 |
| 灵活 (flexibility) | 6 | 需要配合攻击技能使用，单独使用无效 |
| 独特 (uniqueness) | 8 | 下次必定暴击的机制独特且强力 |
| 联动 (synergy) | 10 | 几乎与所有输出技能都有极强联动 |

## 原子指令实现

```typescript
ops: [
  { op: 'set', target: 'self', path: 'variables.crit_bonus', value: 1 },
  { op: 'if', cond: { type: 'compare', path: 'variables.crit_bonus', comparator: '>', value: 0 }, then: [
    { op: 'log', text: '暴击姿态已就绪，下次攻击伤害翻倍' }
  ] },
  { op: 'log', text: '进入暴击姿态，下次攻击造成双倍伤害' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **set**: 设置变量 `variables.crit_bonus` 为 1，标记下一次攻击为暴击状态（变量可通过路径访问，如 variables.crit_bonus）
2. **if**: 条件判断，使用 compare 类型的 cond，当 `variables.crit_bonus > 0` 时，输出暴击就绪的日志提示（if指令的cond类型支持 compare/has_status/no_status/hp_pct/mp_pct/and/or）
3. **log**: 输出技能激活日志，使用 text 参数告知玩家暴击姿态已生效

## 使用场景

- BOSS战关键阶段，配合高威力技能打出爆发
- 敌人血量较低时，配合斩杀阶段快速收尾
- 配合多段攻击技能的第一发，造成高额首击
- 需要快速破盾/破甲后，抓住窗口期爆发输出
- 与力量祝福同时开启，形成爆发组合

## 联动效果

- 与力量祝福联动：攻击力buff + 双倍伤害，输出能力指数级提升
- 与吸血攻击联动：双倍伤害带来双倍吸血，瞬间回血量惊人
- 与疾风步联动：高速可以更快地打出暴击攻击
- 与反噬护盾联动：护盾保障生存，确保能稳定等到爆发窗口
