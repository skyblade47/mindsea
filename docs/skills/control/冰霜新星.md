# 冰霜新星

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 控制 |
| 伤害类型 | 魔法 |
| 使用次数 | 3 次 |
| 品质 | 钢语 |
| 基础威力 | 60 |
| MP消耗 | 18 |

## 技能描述

释放冰霜能量形成新星爆发，对敌人造成冰属性魔法伤害的同时，寒气会侵入敌人体内降低其移动速度和攻击速度。被减速的敌人行动顺序延后，出手频率降低，能够有效削弱敌方的输出能力。冰霜新星兼具伤害与控制，是非常实用的多功能技能。

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'enemy', path: 'hp', value: -60 },
  { op: 'clamp', target: 'enemy', path: 'hp', min: 0, max: 999 },
  { op: 'mul', target: 'enemy', path: 'speed', value: 0.6 },
  { op: 'set', target: 'enemy', path: 'slow_duration', value: 3 },
  { op: 'clamp', target: 'enemy', path: 'speed', min: 10, max: 999 },
  { op: 'log', text: '冰霜能量爆发！寒气侵入敌体，速度大幅下降！' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add - 造成伤害**: 对敌人造成 60 点冰属性魔法伤害，生命值减少 60。
2. **clamp - 限制生命下限**: 确保敌人生命值不会低于 0，防止出现负数血量。
3. **mul - 降低敌人速度**: 将敌人的速度乘以 0.6，即降低 40% 的速度，实现减速效果。
4. **set - 设置减速持续时间**: 设置减速效果的持续时间为 3 回合，3 回合后速度恢复正常。
5. **clamp - 限制速度下限**: 确保敌人速度不会低于 10，防止速度过低导致无法行动。
6. **log - 记录技能日志**: 输出冰霜新星释放的提示信息。

## 使用场景

- 面对高速敌人时使用，降低其出手频率
- BOSS战中配合队友集火输出
- 需要控制战场节奏时使用
- 与冰系队友配合触发连锁效果

## 联动效果

- 与「冰霜护甲」联动：双冰系技能组合，可能触发冰系专精加成
- 与「眩晕打击」联动：减速+眩晕双重控制，敌人几乎无法行动
- 与「沉默咒文」联动：减速+降攻，全方位削弱敌方输出能力
- 与「生命回复」联动：控制敌人的同时保持自身血量，稳扎稳打
