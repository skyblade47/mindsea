# 眩晕打击

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 控制 |
| 伤害类型 | 物理 |
| 使用次数 | 2 次 |
| 品质 | 银句 |
| 基础威力 | 45 |
| MP消耗 | 20 |

## 技能描述

集中力量攻击敌人的要害部位，有较高几率使敌人陷入眩晕状态。被眩晕的敌人在1回合内无法行动，只能任由我方宰割。虽然伤害不高，但控制效果极强，往往能在关键时刻扭转战局。是性价比极高的控制技能。

## 六维评估

| 维度 | 评分 | 说明 |
|------|------|------|
| 威力 (power) | 4 | 直接伤害较低，但眩晕效果价值极高 |
| 范围 (scope) | 3 | 单体控制技能，作用范围有限 |
| 持续 (duration) | 6 | 眩晕持续1回合，但使用次数较少 |
| 灵活 (flexibility) | 7 | 几乎所有战斗都能发挥作用 |
| 独特 (uniqueness) | 8 | 强控制效果，机制较为稀有 |
| 联动 (synergy) | 8 | 与高爆发技能配合效果极佳 |

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'enemy', path: 'hp', value: -45 },
  { op: 'clamp', target: 'enemy', path: 'hp', min: 0, max: 999 },
  { op: 'random', min: 0, max: 100, assignTo: 'stun_chance' },
  { op: 'if', cond: { type: 'compare', path: 'stun_chance', comparator: '<', value: 75 }, then: [
    { op: 'set', target: 'enemy', path: 'stunned', value: 1 },
    { op: 'set', target: 'enemy', path: 'stun_duration', value: 1 },
    { op: 'log', text: '眩晕成功！敌人陷入眩晕状态！' }
  ], else: [
    { op: 'log', text: '眩晕失败，敌人抵抗了控制效果。' }
  ]}
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add - 造成基础伤害**: 对敌人造成 45 点物理伤害，作为技能的基础伤害部分。
2. **clamp - 限制生命下限**: 确保敌人生命值不会低于 0，防止出现负数血量。
3. **random - 随机眩晕判定**: 生成一个 0 到 100 的随机数，赋值到 stun_chance，用于判定眩晕是否成功。
4. **if - 条件判断**: 使用 compare 类型条件，当 stun_chance 小于 75 时（即 75% 几率），触发眩晕效果；否则眩晕失败。
   - 成功分支：设置敌人的眩晕状态为 1（已眩晕），持续时间为 1 回合，并记录成功日志。
   - 失败分支：记录眩晕失败的日志。

## 使用场景

- 面对高伤害敌人时，眩晕使其无法出手
- BOSS战的关键阶段控制BOSS
- 配合高爆发队友，在眩晕期间集火秒杀
- 残血时控制敌人争取喘息时间
- PVP对战中限制敌方核心输出

## 联动效果

- 与「铁壁」联动：控住敌人后开启防御，稳如泰山
- 与「冰霜新星」联动：减速+眩晕双重控制，敌人完全失去行动能力
- 与「沉默咒文」联动：眩晕+沉默，敌人既不能动也不能放技能
- 与「生命回复」联动：控住敌人后慢慢回血，以战养战
