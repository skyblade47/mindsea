# 沉默咒文

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 控制 |
| 伤害类型 | 魔法 |
| 使用次数 | 2 次 |
| 品质 | 银句 |
| 基础威力 | 30 |
| MP消耗 | 25 |

## 技能描述

施展神秘的封禁咒文，使敌人陷入沉默状态。被沉默的敌人无法使用任何技能，只能进行普通攻击。为了模拟沉默效果，该技能会大幅降低敌人的攻击力，使其普通攻击也变得软弱无力。沉默对于依赖技能输出的敌人来说是毁灭性的打击。

## 六维评估

| 维度 | 评分 | 说明 |
|------|------|------|
| 威力 (power) | 5 | 直接伤害低，但降攻和沉默效果很强 |
| 范围 (scope) | 3 | 单体控制，作用范围有限 |
| 持续 (duration) | 7 | 持续3回合，控制时间较长 |
| 灵活 (flexibility) | 6 | 对技能型敌人效果极佳，对普攻型作用有限 |
| 独特 (uniqueness) | 7 | 沉默机制较为特殊，针对性强 |
| 联动 (synergy) | 6 | 与其他控制技能配合形成控制链 |

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'enemy', path: 'hp', value: -30 },
  { op: 'clamp', target: 'enemy', path: 'hp', min: 0, max: 999 },
  { op: 'mul', target: 'enemy', path: 'attack', value: 0.3 },
  { op: 'set', target: 'enemy', path: 'silenced', value: 1 },
  { op: 'set', target: 'enemy', path: 'silence_duration', value: 3 },
  { op: 'clamp', target: 'enemy', path: 'attack', min: 5, max: 999 },
  { op: 'log', text: '神秘咒文生效！敌人被沉默，攻击力大幅下降！' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add - 造成基础伤害**: 对敌人造成 30 点魔法伤害，作为技能的基础伤害部分。
2. **clamp - 限制生命下限**: 确保敌人生命值不会低于 0，防止出现负数血量。
3. **mul - 降低敌人攻击力**: 将敌人的攻击力乘以 0.3，即降低 70% 的攻击力，模拟沉默后只能普攻的效果。
4. **set - 设置沉默状态**: 将敌人的沉默状态标记为 1，表示敌人已被沉默。
5. **set - 设置沉默持续时间**: 设置沉默效果的持续时间为 3 回合，3 回合后效果消失。
6. **clamp - 限制攻击下限**: 确保敌人攻击力不会低于 5，防止攻击力为 0 或负数。
7. **log - 记录技能日志**: 输出沉默咒文生效的提示信息。

## 使用场景

- 面对依赖技能输出的法师型敌人
- BOSS战中限制BOSS的强力技能
- 敌方有高爆发技能角色时优先沉默
- 配合队友集火，在沉默期内击杀目标
- 应对敌方治疗职业，使其无法施放治疗技能

## 联动效果

- 与「眩晕打击」联动：眩晕+沉默双重控制，敌人完全失去威胁
- 与「冰霜新星」联动：减速+沉默+降攻，全方位削弱敌方
- 与「铁壁」联动：降低敌人输出同时提升自身防御，固若金汤
- 与「生命回复」联动：削弱敌人后慢慢回血，稳操胜券
