# 暗影突袭

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 攻击 |
| 伤害类型 | 物理 |
| 使用次数 | 2 次 |
| 品质 | 银句 |
| 基础威力 | 50 |
| MP消耗 | 20 |

## 技能描述

融入黑暗之中，以迅雷不及掩耳之势向敌人发动致命突袭。这一击凝聚了暗影的力量，造成极高的物理爆发伤害。不仅如此，暗影突袭还有着不俗的暴击几率，一旦触发暴击，伤害将更加恐怖。由于消耗巨大，每场战斗只能使用有限的次数，是关键时刻的绝杀技能。

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'enemy', path: 'hp', value: -50 },
  { op: 'random', target: 'self', min: 0, max: 100, assignTo: 'critChance' },
  { op: 'if', target: 'self', cond: { type: 'compare', path: 'critChance', comparator: '<=', value: 40 }, then: [
    { op: 'add', target: 'enemy', path: 'hp', value: -25 }
  ]},
  { op: 'clamp', target: 'enemy', path: 'hp', min: 0, max: 999 },
  { op: 'log', text: '暗影突袭命中！造成50点物理伤害。' },
  { op: 'if', target: 'self', cond: { type: 'compare', path: 'critChance', comparator: '<=', value: 40 }, then: [
    { op: 'log', text: '暴击！额外造成25点伤害！' }
  ]}
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add hp -50**: 对敌人造成50点基础物理伤害，使用add指令触发护盾和伤害日志
2. **random critChance**: 生成0-100的随机数，赋值给self.critChance，用于判断暴击是否触发
3. **if compare critChance <= 40**: 当随机数小于等于40时（40%暴击率），触发暴击，额外造成25点伤害
4. **clamp hp**: 确保敌人生命值不会低于0（死亡下限）或超过最大生命值上限
5. **log text**: 记录技能命中的基础伤害信息
6. **if compare critChance <= 40**: 如果触发了暴击，额外记录暴击伤害的日志信息

## 使用场景

- **斩杀低血敌人**: 在敌人血量较低时使用，确保将其击杀
- **爆发输出窗口**: 在敌人防御被削弱或有增伤buff时使用，最大化伤害
- **BOSS战关键阶段**: 在BOSS血量进入关键阶段时打出爆发，快速压血线

## 联动效果

- **与「暗影之力」联动**: 提升暴击率至60%，暴击伤害提高至200%
- **与「破甲击」联动**: 敌人防御降低后，暗影突袭的实际伤害大幅提升
- **与「致命偷袭」联动**: 从背后攻击时，暴击伤害额外增加50%
