# 破甲击

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 攻击 |
| 伤害类型 | 物理 |
| 使用次数 | 3 次 |
| 品质 | 铁言 |
| 基础威力 | 18 |
| MP消耗 | 10 |

## 技能描述

集中力量于一点，向敌人的护甲薄弱处发起猛烈一击。虽然直接造成的物理伤害不算很高，但这一击会破坏敌人的防御结构，使其防御力大幅降低。破甲效果可持续多个回合，让后续的物理攻击能够造成更大的伤害。是对付高防御敌人的利器。

| 独特 (uniqueness) | 7 | 减少敌人防御的辅助攻击技能，机制有特色 |
| 联动 (synergy) | 9 | 与所有物理攻击技能都有极强的配合效果 |

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'enemy', path: 'hp', value: -18 },
  { op: 'add', target: 'enemy', path: 'defense', value: -15 },
  { op: 'set', target: 'enemy', path: 'debuff.armorBreak.duration', value: 3 },
  { op: 'clamp', target: 'enemy', path: 'hp', min: 0, max: 999 },
  { op: 'clamp', target: 'enemy', path: 'defense', min: 0, max: 999 },
  { op: 'log', text: '破甲击命中！造成18点物理伤害，敌人防御力下降15点。' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add hp -18**: 对敌人造成18点基础物理伤害，使用add指令触发护盾和伤害日志
2. **add defense -15**: 降低敌人15点防御力，使其受到的物理伤害增加
3. **set debuff.armorBreak.duration**: 设置破甲减益效果持续3回合
4. **clamp hp**: 确保敌人生命值不会低于0（死亡下限）或超过最大生命值上限
5. **clamp defense**: 限制敌人防御力不会低于0，避免出现负防御的情况
6. **log text**: 记录技能施放信息，包括伤害和降防效果

## 使用场景

- **对抗高防御敌人**: 破甲后能显著提升物理攻击的伤害
- **战斗开局**: 先手使用破甲，让队友后续的攻击都能享受增伤
- **物理队配合**: 全队都是物理输出时，破甲的收益最大化

## 联动效果

- **与「重击」联动**: 重击对破甲状态的敌人造成额外50%伤害
- **与「撕裂」联动**: 破甲状态下，撕裂DOT的伤害提高30%
- **与「战吼」联动**: 队友攻击力提升+敌人防御降低，双重增伤效果爆炸
