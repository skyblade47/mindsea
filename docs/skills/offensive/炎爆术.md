# 炎爆术

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 攻击 |
| 伤害类型 | 魔法 |
| 使用次数 | 2 次 |
| 品质 | 钢语 |
| 基础威力 | 35 |
| MP消耗 | 15 |

## 技能描述

凝聚炽热的火焰能量，向敌人投掷一颗爆裂的火球。火球在命中目标后爆炸，造成高额魔法伤害的同时，点燃敌人的身躯，使其在接下来的回合中持续受到灼烧伤害。灼烧效果会随时间逐渐减弱，但足以让敌人在痛苦中挣扎。

| 独特 (uniqueness) | 6 | 经典的火焰DOT技能，机制成熟但不算特别新颖 |
| 联动 (synergy) | 7 | 可与增伤、引爆类技能配合，触发额外效果 |

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'enemy', path: 'hp', value: -35 },
  { op: 'set', target: 'enemy', path: 'dot.burn.value', value: 8 },
  { op: 'set', target: 'enemy', path: 'dot.burn.duration', value: 3 },
  { op: 'clamp', target: 'enemy', path: 'hp', min: 0, max: 999 },
  { op: 'log', text: '炎爆术命中！造成35点魔法伤害，敌人陷入灼烧状态。' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add hp -35**: 对敌人造成35点基础魔法伤害，使用add指令触发护盾和伤害日志
2. **set dot.burn.value**: 设置灼烧DOT伤害值为每回合8点
3. **set dot.burn.duration**: 设置灼烧效果的持续时间为3回合
4. **clamp hp**: 确保敌人生命值不会低于0（死亡下限）或超过最大生命值上限
5. **log text**: 记录技能施放信息，用于战斗日志显示

## 使用场景

- **BOSS战**: 对高血量BOSS使用，利用DOT持续消耗其生命值
- **持久战**: 在战斗初期使用，让灼烧效果在多回合中累计伤害
- **对抗高防御敌人**: 魔法伤害无视部分物理防御，DOT效果可绕过防御减免

## 联动效果

- **与「烈焰增幅」联动**: 提升灼烧DOT的伤害值，使每回合伤害提高50%
- **与「爆燃」联动**: 立即引爆敌人身上的灼烧效果，造成剩余DOT总伤害的爆发伤害
- **与「风刃术」联动**: 风助火势，灼烧伤害额外增加30%
