# 毒刃

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 攻击 |
| 伤害类型 | 物理 |
| 使用次数 | 4 次 |
| 品质 | 铁言 |
| 基础威力 | 20 |
| MP消耗 | 8 |

## 技能描述

将剧毒涂抹在利刃之上，快速刺向敌人。刀刃造成物理伤害的同时，剧毒会渗入敌人的血液，使其陷入中毒状态。毒素会在敌人体内慢慢蔓延，持续造成伤害并逐渐削弱其战斗力。虽然单次伤害不高，但使用次数较多，适合在战斗中频繁使用。

| 持续 (duration) | 8 | 中毒效果持续4回合，是持续时间最长的DOT技能之一 |
| 灵活 (flexibility) | 7 | 使用次数多达4次，适用于各种战斗场景 |
| 独特 (uniqueness) | 5 | 经典的物理中毒技能，机制常见但实用 |
| 联动 (synergy) | 6 | 可与其他毒素技能叠加，或与减益技能配合 |

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'enemy', path: 'hp', value: -20 },
  { op: 'set', target: 'enemy', path: 'dot.poison.value', value: 5 },
  { op: 'set', target: 'enemy', path: 'dot.poison.duration', value: 4 },
  { op: 'clamp', target: 'enemy', path: 'hp', min: 0, max: 999 },
  { op: 'clamp', target: 'enemy', path: 'dot.poison.value', min: 0, max: 50 },
  { op: 'log', text: '毒刃命中！造成20点物理伤害，敌人中毒了。' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add hp -20**: 对敌人造成20点基础物理伤害，使用add指令触发护盾和伤害日志
2. **set dot.poison.value**: 设置中毒DOT伤害值为每回合5点
3. **set dot.poison.duration**: 设置中毒效果的持续时间为4回合
4. **clamp hp**: 确保敌人生命值不会低于0（死亡下限）或超过最大生命值上限
5. **clamp dot.poison.value**: 限制毒素伤害叠加上限，防止无限堆叠导致失衡
6. **log text**: 记录技能施放信息，用于战斗日志显示

## 使用场景

- **消耗战**: 战斗开始时使用，让毒素在多回合中持续造成伤害
- **多目标战斗**: 使用次数多，可以给多个敌人分别上毒
- **对抗高恢复敌人**: 中毒持续伤害可以抵消敌人的部分回复效果

## 联动效果

- **与「剧毒涂层」联动**: 提升中毒DOT的伤害值，使每回合毒素伤害提高40%
- **与「虚弱诅咒」联动**: 中毒状态下的敌人受到的所有伤害增加20%
- **与「毒液爆发」联动**: 立即消耗所有中毒层数，造成一次性爆发伤害
