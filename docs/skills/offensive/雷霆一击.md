# 雷霆一击

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 攻击 |
| 伤害类型 | 魔法 |
| 使用次数 | 4 次 |
| 品质 | 钢语 |
| 基础威力 | 30 |
| MP消耗 | 12 |

## 技能描述

召唤天雷之力，将雷电凝聚于武器之上，猛地向敌人劈下。雷霆一击不仅造成可观的雷电魔法伤害，还有几率使敌人陷入眩晕状态，让其在一回合内无法行动。雷电的速度极快，往往让敌人来不及反应就被击中。使用次数较多，是兼具输出与控制的实用技能。

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'enemy', path: 'hp', value: -30 },
  { op: 'random', target: 'self', min: 0, max: 100, assignTo: 'stunChance' },
  { op: 'if', target: 'self', cond: { type: 'compare', path: 'stunChance', comparator: '<=', value: 35 }, then: [
    { op: 'set', target: 'enemy', path: 'stun', value: 1 }
  ]},
  { op: 'clamp', target: 'enemy', path: 'hp', min: 0, max: 999 },
  { op: 'log', text: '雷霆一击命中！造成30点雷电伤害。' },
  { op: 'if', target: 'enemy', cond: { type: 'compare', path: 'stun', comparator: '>', value: 0 }, then: [
    { op: 'log', text: '敌人被眩晕了！下回合无法行动。' }
  ]}
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add hp -30**: 对敌人造成30点基础雷电魔法伤害，使用add指令触发护盾和伤害日志
2. **random stunChance**: 生成0-100的随机数，赋值给self.stunChance，用于判断眩晕是否触发
3. **if compare stunChance <= 35**: 当随机数小于等于35时（35%概率），触发眩晕效果
4. **clamp hp**: 确保敌人生命值不会低于0（死亡下限）或超过最大生命值上限
5. **log text**: 记录技能命中的基础伤害信息
6. **if compare stun > 0**: 检查敌人是否被眩晕，如果是则额外记录眩晕日志

## 使用场景

- **对抗高威胁敌人**: 眩晕可以让敌人跳过一回合，大幅降低受到的伤害
- **BOSS战**: 在BOSS释放大招前使用，有几率打断其关键技能
- **持久战**: 使用次数多，可以频繁尝试触发眩晕，积累控制优势

## 联动效果

- **与「雷电增幅」联动**: 提升雷电伤害，基础威力提高25%
- **与「麻痹毒药」联动**: 眩晕成功时，额外给敌人附加中毒效果
- **与「集火」联动**: 队友对眩晕状态的敌人造成的伤害提高30%
