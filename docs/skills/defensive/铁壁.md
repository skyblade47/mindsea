# 铁壁

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 防御 |
| 伤害类型 | 物理 |
| 使用次数 | 2 次 |
| 品质 | 银句 |
| 基础威力 | 85 |
| MP消耗 | 22 |

## 技能描述

催动内劲使全身肌肉骨骼如钢铁般坚硬，大幅提升自身防御力。在效果持续期间，受到的物理伤害将大幅降低，是面对物理攻击型敌人的强力防御技能。铁壁状态下的角色宛若铜墙铁壁，令敌人的攻击难以奏效。

## 原子指令实现

```typescript
ops: [
  { op: 'mul', target: 'self', path: 'defense', value: 2.5 },
  { op: 'clamp', target: 'self', path: 'defense', min: 0, max: 300 },
  { op: 'set', target: 'self', path: 'defense_buff_duration', value: 2 },
  { op: 'log', text: '全身肌肉紧绷如钢铁，防御力大幅提升！' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **mul - 倍率提升防御力**: 将自身防御力乘以 2.5 倍，实现大幅防御提升的效果。
2. **clamp - 限制防御上限**: 将防御力限制在 0 到 300 之间，防止防御数值过高导致失衡。
3. **set - 设置持续时间**: 设置防御增益的持续时间为 2 回合，2 回合后效果自动消失。
4. **log - 记录技能日志**: 输出铁壁激活的提示信息。

## 使用场景

- 面对高物理伤害的BOSS时使用
- 充当队伍前排吸引火力时开启
- 预判敌人即将发动物理大招前使用
- 与敌方物理输出型角色单挑时

## 联动效果

- 与「冰霜护甲」联动：双重防御叠加，物理减伤达到极致
- 与「生命回复」联动：高防御配合持续回血，生存能力极强
- 与「眩晕打击」联动：先控后防，掌握战斗节奏
