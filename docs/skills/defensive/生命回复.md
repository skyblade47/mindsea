# 生命回复

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 防御 |
| 伤害类型 | 魔法 |
| 使用次数 | 3 次 |
| 品质 | 钢语 |
| 基础威力 | 55 |
| MP消耗 | 12 |

## 技能描述

施放宽和的治愈魔法，使自身获得持续回复生命的效果。在接下来的数回合内，每回合开始时都会回复一定量的生命值。该技能虽然单次回复量不高，但胜在持续时间长，能够在持久战中有效维持血量，是非常实用的续航技能。

## 六维评估

| 维度 | 评分 | 说明 |
|------|------|------|
| 威力 (power) | 5 | 单次回复量一般，但总回复量可观 |
| 范围 (scope) | 2 | 仅作用于自身，范围较小 |
| 持续 (duration) | 8 | 持续4回合，持续时间长 |
| 灵活 (flexibility) | 8 | 几乎所有战斗都能用，泛用性极强 |
| 独特 (uniqueness) | 4 | 持续回血是经典机制，较为常见 |
| 联动 (synergy) | 7 | 与各种防御技能配合都能产生不错效果 |

## 原子指令实现

```typescript
ops: [
  { op: 'set', target: 'self', path: 'dot.regen.value', value: 25 },
  { op: 'set', target: 'self', path: 'dot.regen.duration', value: 4 },
  { op: 'add', target: 'self', path: 'hp', value: 10 },
  { op: 'clamp', target: 'self', path: 'hp', min: 0, max: 999 },
  { op: 'log', text: '温暖的治愈能量流遍全身，生命回复效果已激活！' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **set - 设置回血量**: 设置每回合的生命回复值为 25 点，作为持续回复的基础数值。
2. **set - 设置回复持续时间**: 设置持续回血效果的持续时间为 4 回合，4 回合后效果结束。
3. **add - 立即回复少量生命**: 技能施放时立即回复 10 点生命值，提供即时的血量补充。
4. **clamp - 限制生命上限**: 确保当前生命值不会超过最大生命值 999，防止溢出。
5. **log - 记录技能日志**: 输出生命回复激活的提示信息。

## 使用场景

- 持久战中维持血量稳定
- 低血量时配合防御技能保命
- BOSS战的P2/P3阶段保持续航
- 探索未知区域时预防突发战斗

## 联动效果

- 与「冰霜护甲」联动：护盾减伤配合持续回血，生存能力大幅提升
- 与「铁壁」联动：高防御+持续回血，几乎不会被物理输出打死
- 与「沉默咒文」联动：限制敌方输出的同时慢慢回血，稳扎稳打
