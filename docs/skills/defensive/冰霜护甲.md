# 冰霜护甲

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能类型 | 防御 |
| 伤害类型 | 魔法 |
| 使用次数 | 3 次 |
| 品质 | 钢语 |
| 基础威力 | 65 |
| MP消耗 | 15 |

## 技能描述

召唤冰霜之力包裹自身，形成一层坚固的寒冰护甲。该技能同时为自身增加护盾值并提升防御力，能够有效抵御敌人的物理和魔法攻击。护盾在受到攻击时会优先吸收伤害，而防御力的提升则进一步减少受到的伤害。

## 原子指令实现

```typescript
ops: [
  { op: 'add', target: 'self', path: 'shield', value: 65 },
  { op: 'add', target: 'self', path: 'defense', value: 20 },
  { op: 'clamp', target: 'self', path: 'defense', min: 0, max: 200 },
  { op: 'log', text: '寒冰覆盖全身，防御力提升！' }
]
```

## 指令解析

逐条解释每条原子指令的作用：

1. **add - 增加护盾值**: 为自身增加 65 点护盾，形成一层可吸收伤害的寒冰护盾。
2. **add - 增加防御力**: 在现有防御力基础上增加 20 点，提升物理和魔法伤害的减免效果。
3. **clamp - 限制防御上限**: 确保防御力不会超过 200 的上限，防止数值溢出破坏平衡。
4. **log - 记录技能日志**: 输出技能激活的提示信息。

## 使用场景

- BOSS战中面对高伤害敌人时使用
- 队伍中缺乏治疗职业时的自保手段
- 预判敌人即将发动强力攻击前开启
- 持久战中配合其他防御技能形成多层防护

## 联动效果

- 与「生命回复」联动：在护甲保护下持续回血，大幅提升生存能力
- 与「铁壁」联动：双重防御提升，达到近乎免疫物理伤害的效果
- 与「冰霜新星」联动：冰属性技能组合，可能触发冰系专精加成
