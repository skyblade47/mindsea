# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.1.0] - 2026-06-23

### Added
- Initial project setup with monorepo structure
- Shared types and constants package
- Server with Express and TypeScript
- Client with React, Vite, and TailwindCSS
- Creation Engine (skill generation with fragments)
- Adventure Engine (narrative nodes and progression)
- Combat Engine (turn-based battle system)
- Rule Engine (business logic and validation)
- AI integration layer with Mock and OpenAI support
- Unified Creation System v3.0
- Fragment-based skill crafting
- Ancient Lexicon system (preset skills from collectible fragments)
